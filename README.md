# Notenalarm

Dieses Projekt dient dazu, Nutzer automatisch über neue Noten zu informieren.
Benachrichtigungen erfolgen wahlweise über Telegram oder als Desktop-Push-Mitteilungen.

Grundlage ist ein Telegram-Bot, der in einer Gruppe eingebunden wird und dort alle Mitglieder bei neuen Ergebnissen benachrichtigt.
